Drop project screenshots here, named 1.png, 2.png, 3.png (etc) inside each
project's folder, e.g.:

  public/projects/chatty/1.png
  public/projects/chatty/2.png

Then reference them in src/data/resume.ts under that project's `images` array,
e.g. images: ['/projects/chatty/1.png', '/projects/chatty/2.png']

Recommended: 16:9 screenshots (e.g. 1280x720), 2-4 per project. JPG/PNG/WebP all work.
Until images are added, the card will show a "Screenshots coming soon" placeholder.
